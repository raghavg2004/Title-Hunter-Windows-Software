# Title Hunter Windows Software

Title Hunter is a Windows desktop application distributed through this repository as a ready-to-run installer.

## What This Repo Contains

- A Windows installer for Title Hunter
- The latest build published for direct download and installation

## Download

Download the installer from the repository files:

- [Title-Hunter-setup.exe](Title-Hunter-setup.exe)

## Install

1. Download `Title-Hunter-setup.exe`.
2. Double-click the file to launch the installer.
3. Follow the setup prompts.
4. Open Title Hunter from the Start menu or the shortcut created by the installer.

## Requirements

- Windows 10 or newer
- Permission to run downloaded applications on your device

## Notes

- The installer file is fairly large, so the first download may take a moment.
- If Windows shows a security prompt, confirm that you trust the source before continuing.

## Support

If you run into issues installing or launching the app, check the repository history or release source for the latest build details.
