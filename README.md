Title Hunter is a Windows desktop app that identifies the most likely movie or web series from a YouTube or Instagram link.

## What this app does

- Accepts a YouTube or Instagram video link.
- Reads public metadata from the link.
- Pulls YouTube transcripts when available.
- Shows the result in a compact result dialog with title, media type, and confidence.
- Provides a settings panel for appearance and app information.
- Uses a branded splash screen and loading dialog during startup and analysis.

## Key features


- Glassmorphism-inspired panels and cinematic dark/gold styling.
- Light, dark, and system theme support.
- Startup splash screen with animated progress.
- Loading dialog while the app analyzes a link.
- Result dialog that highlights the detected title and confidence.
- Settings window with Appearance and About tabs.
- Transcript fallback for YouTube videos.

## How it works

1. You paste a public YouTube or Instagram link into the app.
2. The app extracts metadata such as title, description, uploader, and channel details.
3. For YouTube links, it attempts to fetch transcripts.
6. The best match is shown with a confidence score and supporting evidence.

## Requirements

- Windows 10 or Windows 11
- Internet access for link analysis


## Installation

1. Download `Title-Hunter-setup.exe`.
2. Double-click the file to launch the installer.
3. Follow the setup prompts.
4. Open Title Hunter from the Start menu or the shortcut created by the installer.


## Limitations

- This is a best-effort identifier, not a guaranteed match.
- Public links work best.
- Instagram links can be stricter than YouTube links.
- If download or transcript access fails, the app will still try a metadata-only guess.

## UI notes

- The main search area is centered in a Windows-style dashboard.
- The top-right Settings button opens appearance and about controls.
- The result dialog shows the final match in a compact, focused layout.
